## 快捷键

1. nvim term

```vim
" start term in nvim "
:te
:terminal
:e term://$SHELL
:vs term://$SHELL
:split term://$SHELL
:tabe term://$SHELL

" exit term "
<c-\><c-n>

" help "
:help :te
```
